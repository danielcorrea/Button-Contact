Welcome to Joomla Button Contact site.

If you work with Joomla it is possible that at some point need to insert a link to a contact directly within an article or module.

By default Joomla does not include this option. It forces us to copy the URL for a contact from a menu created anteiormente. What causes loss of time and resources.

This plugin eliminates the need for this and offers a menu button next to the text editor that allows the automatic creation of a link to the contact.
